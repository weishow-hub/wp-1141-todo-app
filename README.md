# 待辦事項管理應用程式

前端的待辦事項管理應用程式，使用 HTML、CSS 
和 JavaScript 開發。一個功能完整的待辦事項管理應用程式，包含使用者認證系統，使用 HTML、CSS 和 JavaScript 開發。

## 功能特色

### 使用者認證
- ✅ 使用者註冊（密碼長度至少 4 個字元）
- ✅ 使用者登入
- ✅ 使用者登出
- ✅ 自動記住登入狀態

### 待辦事項管理
- ✅ 新增待辦事項（包含標題和描述）
- ✅ 編輯待辦事項
- ✅ 刪除待辦事項
- ✅ 標記完成狀態（複選框）
- ✅ 點擊待辦事項展開/收起描述
- ✅ 每個使用者的待辦事項獨立儲存

### 使用者體驗
- ✅ 響應式設計
- ✅ 自動滾動條
- ✅ 美觀的 UI 設計
- ✅ 錯誤訊息提示

## 技術架構

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **資料儲存**: localStorage（瀏覽器本地儲存）
- **套件管理**: npm/yarn
- **版本控制**: Git

## 安裝與執行

1. 確保已安裝 Node.js（可選，僅用於套件管理）
2. 克隆專案：
   ```bash
   git clone <repository-url>
   cd wp-1141-todo-app
   ```
3. 在瀏覽器中開啟 `index.html` 檔案

**注意**: 這是一個純前端應用程式，無需安裝依賴或啟動伺服器，直接在瀏覽器中開啟 `index.html` 即可使用。

## 使用說明

### 註冊新帳號
1. 開啟應用程式後，點擊「註冊」連結
2. 輸入使用者名稱（必須唯一）
3. 輸入密碼（至少 4 個字元）
4. 確認密碼
5. 點擊「註冊」按鈕

### 登入
1. 輸入已註冊的使用者名稱
2. 輸入密碼
3. 點擊「登入」按鈕

### 新增待辦事項
1. 登入後，在「新增待辦事項」輸入框中輸入標題
2. 在「描述（選填）」文字框中輸入描述（可選）
3. 點擊「新增」按鈕或按 Enter 鍵新增

### 編輯待辦事項
1. 點擊待辦事項右側的「編輯」按鈕
2. 修改標題和描述
3. 點擊「儲存」按鈕儲存變更，或點擊「取消」取消編輯

### 管理待辦事項
- **點擊待辦事項**: 展開/收起描述
- **複選框**: 標記完成狀態
- **編輯按鈕**: 編輯待辦事項
- **刪除按鈕**: 刪除待辦事項（會顯示確認對話框）

### 登出
點擊右上角的「登出」按鈕即可登出帳號

### 鍵盤快捷鍵
- **Enter**: 在標題輸入框中新增待辦事項
- **Ctrl + Enter**: 在描述輸入框中新增待辦事項
- **Enter**（編輯模式）: 儲存編輯

## 資料儲存

- 所有使用者資料和待辦事項都儲存在瀏覽器的 localStorage 中
- 每個使用者的待辦事項都是獨立儲存的
- 重新整理頁面不會遺失資料（除非清除瀏覽器資料）

⚠️ **重要**: 此應用程式僅供學習和個人使用。由於使用 localStorage 儲存密碼，**不適合用於生產環境**。在實際應用中，應該：
- 使用後端伺服器進行身份驗證
- 使用加密的密碼雜湊（如 bcrypt）
- 使用 HTTPS 傳輸
- 實作適當的安全措施
